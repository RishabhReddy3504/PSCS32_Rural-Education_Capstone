## üîß Troubleshooting

### Problem: Chatbot not opening
**Solution:**
- Scroll to bottom-right corner
- Look for blue bot icon
- Click it to open chat window

### Problem: Slow responses
**Solution:**
- First request: 2-3 seconds (normal)
- Check internet connection
- Try simpler question
- If still slow, fallback will activate

### Problem: Wrong language responses
**Solution:**
- Check language dropdown
- Select correct language
- Try asking in that language
- Settings auto-save

### Problem: Voice not working
**Solution:**
- Check browser permissions (F12 ‚Üí Permissions)
- Allow microphone access
- Allow speaker access
- Refresh page
- Try in different browser

### Problem: Gibberish in response
**Solution:**
- This is very rare
- Simply ask the question again
- System will retry
- If persistent, feedback helps improve
